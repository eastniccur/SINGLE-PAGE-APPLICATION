# WORDLY DICTIONARY.
## INTRODUCTION
- This is a Single Page Application (SPA)  that enables users to search for a word and retrieve information such as pronunciation, definitions, synonyms, and source details.

## FEATURES OF THE SPA
1. Real-time Search: Fetches data from the Free Dictionary API without refreshing the page.

2. Audio Pronunciation: Listen to the correct way to say a word with a built-in audio player.

3. Phonetic Text: Displays phonetic transcriptions (e.g., /bʊk/).

4. Comprehensive Definitions: Provides multiple meanings, parts of speech, and usage examples.

## INSTRUCTIONS ON HOW TO RUN THE PROJECT.
1. Clone the repository to your local machine.

2. Open index.html in your preferred browser (or use VS Code's Live Server).

3. Enter a word into the search bar.

4. Click "Search" or press Enter.

5. Explore the definitions, synonyms, and click the "Play" button to hear the pronunciation

## TECHNOLOGIES USED.
1. HTML5: Semantic structure for the search interface and results display.

2. CSS3: Custom styling focusing on a clean, purple-themed UI for high readability.

3. JavaScript (ES6+):

- Uses the fetch() API with .then() methodology for asynchronous data handling.

- Dynamic DOM manipulation to update content without page reloads.
